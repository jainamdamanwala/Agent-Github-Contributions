# Valid Palindrome

**ID:** 125  
**Difficulty:** Easy  
**URL:** https://leetcode.com/problems/valid-palindrome/  

## Explanation

This problem can be solved by checking if a string is a palindrome or if removing one character from the string makes it a palindrome. The function is_palindrome checks if a string is a palindrome. The main function valid_palindrome checks if a string is a palindrome or if removing one character from the string makes it a palindrome.
