# Validate IP Address

**ID:** 93  
**Difficulty:** Medium  
**URL:** https://leetcode.com/problems/validate-ip-address/  

## Explanation

This problem can be solved by splitting the IP address into four parts and checking each part to see if it is a valid IP address part. A valid IP address part must be between 1 and 3 characters long, cannot start with 0 unless it is 0 itself, and must only contain digits.
