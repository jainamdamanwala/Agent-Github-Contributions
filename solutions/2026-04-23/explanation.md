# Valid Palindrome

**ID:** 125  
**Difficulty:** Easy  
**URL:** https://leetcode.com/problems/valid-palindrome/  

## Explanation

This problem can be solved by counting the frequency of each character in the string. If the string has an odd length, it can only have one character that appears an odd number of times. If the string has an even length, it can have any number of characters that appear an odd number of times.
