# Validate IP Address

**ID:** 678  
**Difficulty:** Medium  
**URL:** https://leetcode.com/problems/validate-ip-address/  

## Explanation

The problem requires us to validate an IP address. We can split the IP address by '.' and check each part. A valid IP address should have four parts and each part should be a number between 0 and 255. We also need to check if each part has at most one leading zero.
