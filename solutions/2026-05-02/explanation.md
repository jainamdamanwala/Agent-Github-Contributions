# Validate IP Address

**ID:** 193  
**Difficulty:** Medium  
**URL:** https://leetcode.com/problems/validate-ip-address/  

## Explanation

This problem can be solved by checking the format of the given string. For IPv4, the string should be divided into four parts by the dot, and each part should be a number between 0 and 255. For IPv6, the string should be divided into eight parts by the colon, and each part should be a hexadecimal number with a maximum length of 4.
