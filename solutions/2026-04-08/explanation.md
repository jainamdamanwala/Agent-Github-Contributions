# Validate IP Address

**ID:** 93  
**Difficulty:** Medium  
**URL:** https://leetcode.com/problems/validate-ip-address/  

## Explanation

This problem requires checking if a given string is a valid IP address. A valid IP address consists of four parts separated by dots, and each part should be a number between 0 and 255. The function splits the input string into parts, checks each part, and returns "Valid" if all parts are valid, otherwise returns "Invalid".
