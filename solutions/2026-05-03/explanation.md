# Valid Palindrome

**ID:** 125  
**Difficulty:** Easy  
**URL:** https://leetcode.com/problems/valid-palindrome/  

## Explanation

This problem can be solved by counting the frequency of each character in the string. A string can be rearranged into a palindrome if and only if at most one character has an odd count. This is because in a palindrome, all characters except possibly one must appear in pairs.
