# Validate IP Address

**ID:** 191  
**Difficulty:** Medium  
**URL:** https://leetcode.com/problems/validate-ip-address/  

## Explanation

This problem can be solved by checking the IP address according to the IPv4 or IPv6 rules. For IPv4, each part should be a number between 0 and 255, and the length of each part should be between 1 and 3. For IPv6, each part should be a hexadecimal number with a length between 1 and 4.
